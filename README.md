AI Assignment - A VOICE-BASED VIRTUAL ASSISTANT FOR MULTILINGUAL CUSTOMER SUPPORT
This project aims to develop an AI-powered voice assistant that can understand customer queries in multiple languages and provide automated responses.

Features:
- Speech to Text Conversion
- Language Detection
- Natural Language Processing
- Text to Speech Response
- Multilingual Customer Support
